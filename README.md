# Responsive Product Landing Page

This project is a modern and responsive product landing page built using HTML and CSS. It is designed with a clean layout, user-friendly interface, and professional footer section.

## 🔥 Features

- Responsive design (mobile-friendly)
- Clean and modern UI
- Navigation bar with links
- Hero section with call-to-action
- Features section with cards
- Professional footer with contact details

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📱 Responsive Design

The layout adapts to different screen sizes using CSS media queries.

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
